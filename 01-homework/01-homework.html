Basic commands

  • git config --global user.name blabla
    • Встановлює юзернейм для автора комітів

  • git config --global user.email blabal@bla
    • Встановлює імейл для автора комітів

  • git init
    • Створює пустий Git репозиторій або реініціалізує існучий і

  • git status
    • Показує статус робочого каталогу

  • git add
    • Додає вміст файлу до індексу

  • git commit
    • Записує зміни в репозиторій

  • git log
    • Показує логи комітів

  • git checkout
    • Команда checkout використовується для копіювання файлів з історії в робочу директорію. Також вона може використовуватися для перемикання між гілками.

  • git branch
    • Показує список гілок. Дає можливість видаляти і створюати гілки

  • git push
    • Публікація нових локальних змін у віддалений репозиторій

  • git pull
    • Завантажує та об'єднює коміти з віддаленого репозиторію в локальний

  • git merge
    • Об'єднює дві або більше гілок разом

  • git clone
    • Клонує репозиторій у нову директорію
